# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-01

### Added
- Initial release of Geni SDK
- OAuth 2.0 authentication module
- Profiles API (get, update, search, family tree)
- Unions API (get, create, update, delete marriages/partnerships)
- Photos API (get, upload, delete photos)
- Search API (search profiles with filters)
- Rate limiter (5,000 requests/day)
- TypeScript type definitions for all APIs
- Comprehensive error handling
- Configurable logging
- Singleton pattern support
- Full documentation and examples
